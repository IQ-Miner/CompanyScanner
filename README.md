# CompanyScanner
Scan a bar code and gain important Information about the Product and the Company behind it

#in Order to run the application i did the following
OPERATIING ON WINDOWS MACHINE

To make react native work:

https://www.youtube.com/watch?v=oorfevovPWw

1. Install Windows Terminal in Microsoft Store
2. If not present install git
	-> https://git-scm.com/
	-> when installing check "(NEW!) Add a Git Bash Profile to Windows Terminal" when selecting Components in Installer
	-> Override the default branch name for new repos with "main"
3. set git bash as standard in terminal
4. install nodejs and select version 18 https://nodejs.org/en/
	-> in installer check the checkbox for tools for native modules
5. if reboot necessary reboot (i had to)
6. now install react native https://reactnative.dev/docs/environment-setup

npm start in git bash and select run android
or npx react-native start
